# 🌤️ App de clima (Frontend) - Módulo 2

Proyecto para visualizar el clima actual y pronóstico semanal de distintas localidades. Desarrollado para el Módulo 2 de Bootcamp Academy.

## 🚀 Enlace al Repositorio
[Repositorio en GitHub](https://github.com/the-spell-of-mathematics/weather-frontend-m2)

## 🛠️ Tecnologías Utilizadas
* **HTML5 Semántico:** Estructuración limpia y accesible mediante etiquetas como `header`, `nav`, `main` y `footer`.
* **Bootstrap 5 (CDN):** Sistema de grillas *mobile-first* (responsivo), *cards* y barra de navegación.
* **JavaScript (Vainilla):** Manipulación del DOM, eventos de usuario y carga dinámica de datos.

## ⚙️ Cómo ejecutar el proyecto
1. Descarga el archivo `.zip` del proyecto o clona el repositorio: `git clone https://github.com/the-spell-of-mathematics/weather-frontend-m2.git`
2. Descomprime el archivo en tu ordenador.
3. Abre el archivo `index.html` en cualquier navegador web moderno (Chrome, Firefox, Edge). No requiere servidor local para su versión básica, pero se recomienda usar extensiones como *Live Server* en VSCode.
